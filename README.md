# Proof of Purpose

An **on-chain nonprofit** — believed to be the first of its kind — built by **MetaPhysical LLC**.

Proof of Purpose is a wallet-primary, on-chain youth impact protocol on Base. Donors fund a transparent treasury in USDC. Trusted partner orgs (Catalysts) propose bounties. Youth (Champions) complete them and earn `$PURPOSE`, a soulbound token redeemable only at approved local vendors at 1:1 USDC.

## Status

Tax-deductibility for donors is on hold pending federal clarity for on-chain charitable structures (e.g. the **CLARITY Act**). Until then, donations are not represented as tax-deductible.

## Stack

- React 18 · Vite · TypeScript · Tailwind
- Base mainnet (chainId 8453)
- thirdweb (smart wallets + contract reads/writes)
- Lovable Cloud (Postgres + Edge Functions + Storage + Auth)
